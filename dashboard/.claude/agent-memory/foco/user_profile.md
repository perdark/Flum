Solo developer building and operating the Fulmen Empire digital goods store. Sells gaming keys, gift cards, and digital products. Customer base spans both retail (B2C) and wholesale/business (B2B/merchant) buyers. Migrating the sales channel from Telegram to a custom Next.js web storefront backed by the same admin dashboard DB.

Primary payment method is ZainCash (Iraq). Card payments are planned but not yet live. Arabic RTL support is a future need.

**How to apply:** When making UI suggestions, keep in mind that the store will be used by Arabic-speaking customers in Iraq. Avoid overly Western-centric copy. The merchant experience (bulk add to cart, business account, invoices, credit terms) is a first-class feature, not an edge case.
